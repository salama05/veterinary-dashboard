# Salama VET - Veterinary Management System

A comprehensive dashboard for managing a veterinary clinic, including sales, purchases, customers, suppliers, treatments, and inventory.

## Features

- **Sales Management**: Track sales, payments, and customer details.
- **Purchase Management**: Manage inventory by tracking purchases from suppliers.
- **Inventory Control**: Real-time tracking of product stock levels.
- **Treatment Records**: Detailed records of animal treatments.
- **Customer & Supplier Management**: Maintain databases for both customers and suppliers.
- **Dashboard Analytics**: Overview of clinic performance and stock status.

## Technologies Used

- **Frontend**: React, Vite, TypeScript, Vanilla CSS.
- **Backend**: Node.js, Express, TypeScript, MongoDB (Mongoose).

## Getting Started

### Prerequisites

- Node.js
- MongoDB

### Installation

1. Clone the repository.
2. Install dependencies for both client and server:
   ```bash
   cd client && npm install
   cd ../server && npm install
   ```
3. Set up your environment variables in a `.env` file in the `server` directory.
4. Start the development servers:
   ```bash
   # In the server directory
   npm run dev
   # In the client directory
   npm run dev
   ```
